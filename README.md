# Сущности
1. Provider - Поставщик
2. Category - Категория
3. Product - Товар
4. Status_p - Статус товара (в наличии/не доступен)
5. Status_d - Статус доставки
6. Client - Клиент
7. Order_in - Товары в заказе
8. Order - Заказ
9. Delivery - Доставка
10. Pickup_point - Пункт выдачи заказов
11. Review - Отзыв на товар
